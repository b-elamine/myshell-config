# dotfiles

my terminal setup: oh-my-zsh + powerlevel10k + some plugins

## what's in here

- `.zshrc` - main zsh config
- `.p10k.zsh` - powerlevel10k prompt config
- `.gitconfig` - git identity
- `.dir_colors/dircolors` - solarized dark colors for ls

plugins: zsh-autosuggestions, zsh-syntax-highlighting, sudo, command-not-found

## install

```bash
git clone https://github.com/YOUR_USERNAME/dotfiles.git ~/dotfiles
cd ~/dotfiles
chmod +x install.sh
./install.sh
```
